<<日本語>>
本アプリケーション EBSD.exe を実行する前に、ランタイムフォルダの
　(1) MicrosoftEdgeWebview2Setup.exe
　(2) VC_redist.x64.exe
を実行してください。

(1)は、本アプリケーションからMicrosoft Edgeブラウザーの機能を呼び出すのに必要なものです。
Windows11では標準でインストールされているので実行する必要はありません。

(2)は、Visual Studio 2019等がインストールされていないマシンで必要になります。

<<English>>
Before running this application EBSD.exe, run
(1) MicrosoftEdgeWebview2Setup.exe
(2) VC_redist.x64.exe
in the runtime folder.

(1) is required to call the Microsoft Edge browser functions from this application.
    It is installed as standard on Windows 11, so there is no need to run it.
(2) is required on machines where Visual Studio 2019 etc. is not installed.
